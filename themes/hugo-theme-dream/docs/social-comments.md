# Social Comments

Now, the dream theme only supports [Disqus](https://disqus.com/) and [utterances](https://utteranc.es/).

## Disqus

In `config.toml`, set:

```toml
disqusShortname = "..."
```

Disqus comments will be showed on the **Single Post Page** and **About Page**.
